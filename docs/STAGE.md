# Project Stage

## Current Stage
ACP is currently in **Introduction, Planning, and Validation**.

This repository is primarily for:
- Publicly defining the protocol idea and primitives
- Gathering design feedback from builders and researchers
- Validating the schema model and state machine boundaries

## What Exists Today
- Draft protocol narrative and principles
- Draft JSON schemas in [`/acp/schemas`](/acp/schemas)
- Early reference artifacts for discussion and validation

## What Does Not Exist Yet
- Production-grade settlement engine
- Formal compliance certification process
- Stable v1.0 backward compatibility guarantees
- Fully specified `VerityReceipt`

## Why This Matters
Early clarity reduces protocol churn later. The goal now is to pressure-test assumptions before ecosystem lock-in.
